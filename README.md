## 📌 Project Overview

This project is an interactive **Sales Dashboard** built in **Microsoft Excel** using the Superstore dataset. The dashboard helps analyze sales performance, profitability, customer trends, and regional performance through Pivot Tables, KPIs, Slicers, and Charts.

The objective of this project is to demonstrate Excel data analysis and dashboard creation skills commonly used in business reporting.

---

## 🚀 Dashboard Features

- 📈 Total Sales KPI
- 💰 Total Profit KPI
- 📦 Total Orders
- 🌍 Regional Sales Analysis
- 🛍️ Category & Sub-Category Performance
- 📅 Year-wise Filtering
- 🎛 Interactive Slicers
- 📊 Pivot Charts
- 📉 Profit Analysis
- 📌 Dynamic Dashboard
